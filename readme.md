# Admin Dashboard Layout

A clean, responsive admin dashboard layout built with **HTML** and **CSS**. It features a complete layout with a **sectioned header**, **sidebar**, and **main content area** – perfect for admin panels or back-office interfaces.

🔗 **Live Demo**: [https://project-admin-dashboard-nine.vercel.app](https://project-admin-dashboard-nine.vercel.app)

## 🧩 Layout Components

The dashboard is structured into three core sections:

- **Sectioned Header** – Contains top navigation, user info (e.g., "Anass Bafakir"), notifications (bell icon), and action buttons ("New Upload", "Share").
- **Sidebar** – Main navigation menu (icons + labels). From your demo, includes a magnifying glass, bell, and user greeting.
- **Main Area** – Dynamic content area where primary data, charts, or tables are displayed.

## 🛠️ Technologies Used

- **HTML5** – Semantic layout structure.
- **CSS3** – Custom styling, Flexbox/Grid for alignment, responsive design.
